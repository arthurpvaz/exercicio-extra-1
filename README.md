Exercício 1: Gerenciamento de Produtos
Crie um programa que permita gerenciar um catálogo de produtos.

Requisitos:

- Cada produto deve ter os atributos: código, nome, preço e quantidade em estoque.
- Use uma classe Produto para representar os produtos.
- Crie uma lista para armazenar os produtos.
- Implemente métodos para:
-- Adicionar novos produtos.
-- Remover produtos pelo código.
-- Atualizar o estoque de um produto específico.
-- Listar todos os produtos cadastrados.

- Desafios adicionais:

- Use um enum para classificar os produtos em categorias como Alimentos, Eletrônicos e Vestuário.
- Trate possíveis exceções, como tentativas de acessar um produto inexistente.
